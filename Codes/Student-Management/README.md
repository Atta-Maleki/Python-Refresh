# 🎓 Student Management System

A simple command-line Student Management System built in Python as a learning project while practicing file handling, functions, loops, conditionals, and basic CRUD operations.

## 📖 Overview

This project stores student information inside a text file (`std.txt`) and allows users to manage student records through an interactive menu.

The application demonstrates how to:

* Read data from files
* Write and update files
* Process user input
* Build menu-driven applications
* Use functions to organize code
* Perform basic data manipulation

---

## 🚀 Features

### 1. View All Students

Displays all student records stored in the file.

### 2. Search Student by ID

Allows the user to enter a student ID and retrieve the corresponding student record.

### 3. Calculate Class Average

Reads all student scores and calculates the class average.

### 4. Add New Student

Adds a new student record to the file with:

* Student ID (generated automatically)
* First Name
* Last Name
* Score

### 5. Edit Student Information

Allows updating:

* First Name
* Last Name
* Score

### 6. Menu-Based Navigation

Provides a simple text-based user interface for interacting with the system.


```bash

```

Run the application:

